# anugopalv.github.io

Personal portfolio website for GitHub Pages, built with only HTML, CSS, and JavaScript.

## Structure

```text
.
|-- index.html
|-- style.css
|-- script.js
|-- images/
|   `-- profile.jpg
`-- README.md
```

## Customize

- Replace `images/profile.jpg` with your profile photo.
- Edit the placeholder text in `index.html`.
- Add your real resume as `resume.pdf` so the resume button works.
- Update the GitHub, LinkedIn, and email links in the Contact section.
- Replace project links with your repository URLs.

## Deploy to GitHub Pages

1. Create a repository named `anugopalv.github.io`.
2. Upload these files to the repository root.
3. Open the repository settings on GitHub.
4. Go to **Pages** and deploy from the `main` branch.
5. Visit `https://anugopalv.github.io`.
